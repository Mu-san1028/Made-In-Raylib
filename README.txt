## コンパイル方法
ディレクトリの構成は次の通りです。

Made-In-Raylib/
├── game.c              # メインのゲームソースコード
├── CMakeLists.txt      # CMake設定ファイル
├── README.txt          # このファイル
├── raylib/             # raylibライブラリのソースコード（グラフィックス/音声ライブラリ）
├── resources/          # ゲームのリソース（画像・音声ファイル）
│   ├── *.png          # 画像ファイル（heart1.pngなど）
│   ├── *.wav          # 効果音ファイル
│   └── *.mp3          # BGM・音声ファイル
└── build/              # ビルド出力ディレクトリ
    ├── game           # コンパイル済み実行ファイル
    └── Makefile       # 生成されたMakefile

・Made-In-Raylibディレクトリ内にbuildディレクトリがない場合に限り、次のコマンドを実行してください。
cd Made-In-Raylib (すでにこのディレクトリにいる場合は実行しなくてOK)
mkdir build
cd build
cmake ..
make

・Made-In-Raylibディレクトリにいる状況で、次の方法でコンパイルしてください。Githubにおいてあるbuildディレクトリではスコアが63から始まる等の不具合が発生する場合があります。
　その場合は、buildディレクトリを消去して上のコマンドを実行してください。
cd build && make && cd .. && ./build/game
または
./build/game

## ゲームのやり方
起動した後、SPACEキーを押すことでゲームがスタートします。ゲームには次の10種類があります。

(1)連打ゲーム
SPACEキーを連打してください。制限時間内に指定された回数押されればクリアです。

(2)タイピングゲーム
指定された単語をタイピングしてください。制限時間内に入力しきればクリアです。

(3)コインゲーム
画面上に表示されるコインをマウスなどでクリックしてください。制限時間内にすべてのコインが取得できればクリアです。

(4)マインスイーパーのゲーム
画面にマインスイーパーの盤面が表示されます。灰色になっているマスのうち、一つが安全なマス、残り全ては爆弾マスです。安全マスを見つけてマウスなどでクリックしてください。

(5)計算ゲーム
画面上に計算問題が表示されます。キーボードで答えを入力してください。制限時間内に指定された問題数解答できればクリアです。

(6)巻物読み取りゲーム
画面左に巻物が表示されます。マウスなどで左から右にドラッグ操作をすることで巻物を読むことができます。制限時間内に巻物を100％読み切ればクリアです。

(7)的あてゲーム
紅白の的が不規則に移動します。ENTERキーを押すことで画面下の中央から上方向に球が発射されます。球が的に当たればクリアです。

(8)6鍵の音ゲーム
画面上に6列枠が表示されます。左からA, S, D, J, K, Lのキーが割り当てられているので、黄色のノーツがあるところに対応するキーを押してください。
なお、灰色のノーツは次に来るノーツの位置を表しています。制限時間内にすべてのノーツを押せればクリアです。

(9)押し分けゲーム
SPACEキーとENTERキーを押し分けるゲームです。小さい丸はSPACEキー、大きい四角はENTERキーに対応しています。黄色のマスがどちらか判別して、対応するボタンを押してください。

(10)神経衰弱ゲーム
画面上にカードが表示されます。神経衰弱の要領でカードをマウスなどでクリックし、全て取り切ってください。

ゲームオーバー後、Rキーを押すことでもう一度最初からやり直すことができます。また、タイピングゲーム以外の時では、Pボタンを押すことでポーズすることができます。


## アピールポイント
　開発が大変過ぎないように、ミニゲームを集めるという方針でゲームを作成した。結果的には、10個ものミニゲームを完成させることができた。
アピールポイントとしては、一つ一つのゲームに最適なSEを割り当てたこと。また、BGMも進行段階に応じて再生速度を変更し臨場感を高めたこと。
演出にもこだわった。例えば、マインスイーパーのゲームではマスをクリックした瞬間に結果が表示されるのではなく、他のマスをすべてオープンすることでより緊張するような演出となった。
計算をするゲームでも、次の問題があらかじめ見えている状態にすることで、考えることを多くしながら難易度を調整しやすくなった。
　また、ゲームが単調な操作で完結しないように、ゲームの種類を多様なものにした。頭を使うもの、記憶するもの、瞬発力が試されたり筋肉が必要なものなどいろいろなものを集めた。
これでゲームに飽きる可能性が少なくなり、より長い時間楽しんでもらえると思っている。
　本ゲームはメイドインワリオをリスペクトしたものであるが、明確に違うのはこれがパソコンで遊べるゲームであるということである。
本家ではボタン操作や回転操作によってミニゲームをクリアしていくが、このゲームはパソコンという特性を生かしてマウスやキーボードを生かさないとクリアできないようなものを多く集めた。
